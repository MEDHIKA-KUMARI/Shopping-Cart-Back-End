# Shopping-Cart-Back-End
A Shopping Cart backend is made using the concepts of Data Structures, OOPs in C++ programming Language. 
This shows the interaction between various entities. The various entities used here are : 
Product class:- Shows anything that we buy from the shopping app. 
Item class:- It will denote the product and the number of products added in the cart. 
Cart:- It represents the collection of items. 
This products includes various methods based on OOPs concepts.
